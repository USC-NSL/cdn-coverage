    A summary of regional PoP coverage for popular CDNs.

    Project maintained by USC-NSL Hosted on GitHub Pages — Theme by mattgraham

    Last updated on 2015/08/27

    This page is an effort to maintain an up-to-date list of what CDNs have PoP locations around the world. Most of the data is compiled directly from the CDN's own public network page, but there is also some data from academic research. We only count distinct PoP metro-level locations (e.g. if a CDN has 2 PoPs in Los Angeles, we only count this as one). Each network name contains a link to the original data source.

    See information about our ongoing CDN work at http://mappinggoogle.cs.usc.edu

    Please send any corrections, updates, etc... to calderm 'at' usc.edu.
